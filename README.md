🌳 Yggdrasil
Every Day Has a Story. Let Yggdrasil Remember It.

Yggdrasil is a modern, secure, and beautifully designed digital journal and productivity platform that combines the power of a diary, planner, calendar, and personal knowledge base into one seamless experience.

Inspired by Notion, Google Calendar, Apple Notes, and the mythical World Tree Yggdrasil, this application helps users organize their thoughts, memories, goals, schedules, and daily life in a private cloud synced workspace.
